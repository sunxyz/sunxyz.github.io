  • 购买域名
    在万网购买一个域名即可

  • 添加域名解析
    以万网为例 添加一个记录类型为CNAME类型的的解析记录 主机记录填写www记录值填YOUR-GITHUB-USERNAME.github.io。（以本网站为例 sunxyz.github.io）一般情况下等待10分钟之后即可访问了，我们可以继续往下进行

  • 创建CNAME文件
    在github仓库中创建一个CNAME文件里面写上刚才购买的域名即可 以我的网站为例
